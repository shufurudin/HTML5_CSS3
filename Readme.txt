Atalhos VS Code:
Ctrl + Shift + P > Emmet: Wrap with Abbreviation (Para atribuir uma tag aos textos selecionados)
Ctrl + K (para escluir uma tag, bastando estar com o "tracinho de escrever" dentro da tag)
Ctrl + Enter (para pular uma linha sem levar o fim da tag junto)
Ctrl + Espaço (abrirá uma janela para navegar dentro da pasta onde está o arquivo sendo desenvolvido, util para imagens)
Shift + TAB (Selecionando a palavra/texto e executando esse comando, ele joga o texto pra trás. O oposto ocorre quando é feito apenas com o TAB)
Alt + selecionar (clicar em multiplos lugares segurando a tecla Alt, fará com que edite em multiplos lugares)
Alt + seta pra cima (volta a pasta no explorador de arquivo)
Alt + Shift + F (formata o código)
Ctrl + ; (atalho para comentar a linha)
