Principais formatações

Negrito/Destaque

Não semantico: Esta é a forma não semantica de negritar uma palavra, com a tag <b>.

Semantico: Esta é a forma semantica de negritar uma palavra, com a tag <strong>.

Itálico/Ênfase

Não semantico: Aqui a palavra está em itálico, com a tag <i>.

Semantico: Agora temos a palavra em ênfase, com a tag <em>.

Texto marcado

Com style local: Texto marcado com a tag <mark> e a cor definida pelo parametro style dentro da própria tag <mark>.

Com style do head: Texto marcado com a tag 'mark', porém a cor é definida pelo style dentro da tag head.

Texto exluído

Texto inserido

Não semantico: Essa é a forma de sublinhar o texto, porém não semanticamente com a tag <u>.

Semantico: Aqui sim a forma correta de sublinhar o texto, de forma semantica com a tag <ins>.
