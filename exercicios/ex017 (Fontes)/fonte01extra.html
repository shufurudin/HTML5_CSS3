Peso

Peso é a espesura da fonte, corriqueiramente conhecido como bold e light por exemplo. A propriedade CSS para alterar o peso da fonte é:

font-weight: ;

Muitas fontes possuem somente o seu peso normal e o bold (nergrito), entretanto a variedade de pesos vai além disso. Os tipos de valores a serem inseridos na propriedade font-weight: são duas: númerica e descritiva.

Peso númerico

Essa é a forma mais minuciosa de selecionar o peso da fonte, indo de 100 em 100 e partindo do 100 até 900. Pouquíssimas fontes tem todas as variações de 100 até 900, então muito provávelmente você irá se deparar com nenhuma mudança no peso caso você saida do 200 para o 300, por exemplo.

Peso descritivo

Esta forma de determinar o peso da fonte é mais genérica e acertiva. São quatro valores possíveis:

  • lighter
  • normal
  • bold
  • bolder

Estilo

O estilo de fonte nada mais é que o itálico, sublinado, riscado e etc. Temos duas formas de "estilizar" nossas fontes:

  • font-style: ;
  • text-decoration: ;

Os valores para estas propriedades são:

  • font-style:
    • italic
    • oblique
    • normal
    • inherit
    • initial
    • unset
  • text-decoration:
    • dashed
    • dotted
    • double
    • line-through
    • none
    • overline
    • solid
    • underline
    • wavy

Mais uma vez, nem todas fontes possuem todas estas variações.

Combinação

Todas essas propriedades são cumulativas, desde que a fonte permita.

Shorthand/mãozinha

Quatro configurações para fontes são utilizadas diariamente na rotina de um desenvolvedor HTML/CSS, são elas:

  • font-style:
  • font-variant:
  • font-weight:
  • font-size:
  • font-family:

Ao invés de digitar estas quatro propriedades em cada declaração, nós podemos utilizar um atalho que damos o nome de shorthand. Nada mais é que uma mãozinha em que o CSS nos proporciona.

Para isso, basta digitar a propriedade font: dentro da declaração. Mas não só isso, os valores que ali serão inseridos tem uma ordem fixa que deve ser obedecida:

  1. font-style
  2. font-weight
  3. font-size
  4. font-family

Vejamos um exemplo:

        
            h1 {
            font: italic bolder 3em 'Work Sans', sans-serif;
            }
